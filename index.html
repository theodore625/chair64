Welcome to Chair64

Chair64 is a new programming platform that runs `.c64s` files.

Explore our features and get started with Chair64 today!

Chair64 Functions

1. log()

Logs output to the console. Accepts strings or numbers as arguments.

2. void(0)

Acts like a 'pass' statement in Python. It does nothing but returns nothing.

3. void(1)

Checks if the process has finished. Returns a boolean indicating whether the finalization is running.

4. void({float})

Used for finalization. Returns nothing but performs necessary cleanup when 'void(0)' is called.

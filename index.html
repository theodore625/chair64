Overview

Chair64 is a virtual platform that runs `.c64s` files. It's designed to be simple yet powerful, allowing you to easily run and test code in a variety of environments.

Whether you're working on your first script or developing complex applications, Chair64 is here to help you get things done quickly and efficiently.

Functions

Here are the functions available in Chair64:

log()

The log() function logs various types of data to the console. You can log text, numbers, booleans, random booleans, or lottery results. Here are some examples:

  • log('Hello, world!') - Logs a string message.
  • log(123) - Logs a number.
  • log(True) - Logs a boolean value (True).
  • log(False) - Logs a boolean value (False).
  • log(randomboolean) - Logs a random boolean value (True or False).
  • log(lottery) - Logs the result of a lottery (Win or Lose).
  • log('Hello, ' "World!") - Logs joined strings and variables.

randomboolean

The randomboolean function returns either True or False randomly. Example:

  • randomboolean - Returns a random boolean value (True or False).

lottery

The lottery function returns a random outcome of either "Win" or "Lose." Example:

  • lottery - Returns a random lottery result ("Win" or "Lose").

void()

The void() function can be used as a placeholder or to check for process finalization. It doesn't do anything but can be useful in certain scenarios:

  • void(0) - Does nothing, useful for placeholders.
  • void(1) - Checks if the process is finalizing and returns a boolean.
  • void(0.0) - Used for cleanup or finalization.

help

The help function displays a list of available commands and their usage. It can be used at any time to see what commands are available:

  • help - Displays the help menu with available commands.

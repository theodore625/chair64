Overview

Chair64 is a virtual platform that runs `.c64s` files. It's designed to be simple yet powerful, allowing you to easily run and test code in a variety of environments.

Whether you're working on your first script or developing complex applications, Chair64 is here to help you get things done quickly and efficiently.

Chair64 Functions

1. log()

Logs output to the console. Accepts strings or numbers as arguments.

2. void(0)

Acts like a 'pass' statement in Python. It does nothing but returns nothing.

3. void(1)

Checks if the process has finished. Returns a boolean indicating whether the finalization is running.

4. void({float})

Used for finalization. Returns nothing but performs necessary cleanup when 'void(0)' is called.
